====================
API の構成要素と規則
====================

--------------
API の構成要素
--------------

サービスコール
--------------

上位階層のソフトウェアから、下位階層のソフトウェア機能を呼び出すためのインターフェース。
特に、カーネルのサービスコールをシステムコールと呼ぶ場合もある。


コールバック
------------

下位階層のソフトウェアから、上位階層のソフトウェアを呼び出すインターフェース。


静的 API
--------

システムコンフィギュレーションファイル中に記述するインターフェース。
オブジェクトの生成情報や初期状態などを定義する。


構成マクロ
----------

下位階層のソフトウェアに関する各種情報を取り出すために使われるマクロ。
